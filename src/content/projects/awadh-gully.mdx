## Overview

A visually rich, elegant website for Awadh Gully — a restaurant specializing in Awadhi (Lucknowi) cuisine. Designed to capture the warmth and authenticity of the dining experience through immersive imagery and smooth interactions.

## Key Features

- **Interactive Menu**: Categorized dishes with descriptions and pricing
- **Online Reservations**: Table booking form with date/time selection
- **Visual Storytelling**: Full-width hero sections with food photography
- **Responsive Design**: Tailwind CSS-powered layout that works on all devices
- **Social Integration**: FontAwesome icons linking to social media profiles

## Design Approach

The design draws inspiration from traditional Awadhi aesthetics — warm golds, deep reds, and earthy tones — combined with modern web conventions. The focus is on creating an appetite-driven visual experience that converts visitors into diners.

## Technical Details

- Built with Tailwind CSS via CDN for rapid styling
- FontAwesome for social media and navigation icons
- Mobile-first responsive layout
- Optimized for fast loading and SEO
