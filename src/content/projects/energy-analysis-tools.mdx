## Overview

A growing collection of financial analysis tools purpose-built for the renewable energy sector. These tools help evaluate project viability, model tariff structures, and run sensitivity analyses for solar and hybrid power plants.

## Planned Features

- **IRR Calculator**: Internal rate of return modeling for solar projects with customizable assumptions
- **Tariff Analyzer**: Compare competitive bidding tariffs across Indian states
- **Sensitivity Engine**: Monte Carlo simulations for key financial variables
- **Debt Sizing Tool**: Automated debt structuring based on DSCR and P90 generation forecasts

## Current Progress

- Core IRR and tariff comparison modules are functional
- Working on integrating PVsyst data import for automated generation profiles
- Sensitivity analysis framework under development
