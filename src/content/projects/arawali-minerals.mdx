## Overview

A professional corporate website built for ARAWALI GROUP, a leading provider of premium industrial minerals in Rajasthan, India. Features a modern, responsive design with zero external framework dependencies.

## Key Features

- **Responsive Layout**: Optimized for desktop, tablet, and mobile with breakpoints at 768px and 1024px
- **Dark/Light Mode**: Automatic theme switching based on user preference
- **SEO Optimized**: Meta tags, structured data, and semantic HTML
- **Zero Dependencies**: Pure HTML, CSS, and vanilla JavaScript — no frameworks needed
- **Security Headers**: XSS protection and content type options

## Design System

- **Color Palette**: Teal (#21808D) primary with warm brown (#5E5240) secondary
- **Typography**: FKGroteskNeue with Berkeley Mono for code elements
- **Components**: Buttons, cards, forms, and navigation with consistent styling

## Sections

- Home — Hero with company overview and featured services
- About — Mission, vision, and core values
- Products — Comprehensive service offerings and quality standards
- Sustainability — Environmental commitment and responsible practices
- Contact — Contact form and business information
