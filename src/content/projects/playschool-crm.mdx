## Overview

A comprehensive SaaS CRM built specifically for playschools and early education centers. Handles everything from student enrollment to attendance tracking, fee management, and parent communication.

## Key Features

- **Student Management**: Complete profiles with enrollment tracking and progress notes
- **Attendance System**: Daily attendance marking with reports and analytics
- **Fee Management**: Automated fee schedules, payment tracking, and receipt generation
- **Parent Portal**: Communication channel between teachers and parents
- **Admin Dashboard**: Overview metrics, staff management, and system configuration

## Technical Stack

- **Frontend**: Next.js with TypeScript and Shadcn/UI component library
- **Backend**: Supabase for authentication, database, and real-time subscriptions
- **ORM**: Prisma for type-safe database queries
- **Styling**: Tailwind CSS with responsive design
- **Deployment**: Vercel with Supabase cloud backend
